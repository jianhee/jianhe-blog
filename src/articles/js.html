    简介

    什么是 JavaScript？

    • 轻量级的脚本语言：JavaScript 不需要编译，解析器读一行执行一行。也不具备开发应用程序的能力，但能操控其它大型程序，比如浏览器。
    • 嵌入式语言：JavaScript 本身提供的核心语法不多，只能用来做一些数学和逻辑运算。也不提供任何与 I/O（输入/输出）相关的 API，只能嵌入其它大型应用程序环境，调用宿主环境（host）提供的底层 API，比如浏览器环境和服务器环境（即 Node 项目）。
    • 对象模型语言：JavaScript 从语法角度看是一种 “对象模型” 语言。宿主环境通过这个模型，描述自己的功能和操作接口，从而通过 JavaScript 控制这些功能。

    语法构成

    JavaScript 的核心语法相当精简，只包括两个部分：

    1. 基本的语法构造：比如操作符、控制结构、语句。
    2. 标准库：一系列具有各种功能的对象，比如 Array、Date、Math 等。

    宿主环境

    JavaScript 的宿主环境提供额外的 API 以便它调用，这种 API 只能在该环境使用，以浏览器为例可以分成三类：

    1. 浏览器控制类：操作浏览器。
    2. DOM 类：操作网页的各种元素。
    3. Web 类：实现互联网的各种功能。

    表达式

    定义

    1. 行：JavaScript 的执行单位，程序从上到下逐行执行。一般情况下一行就是一个语句。
    2. 语句：为了完成某种任务而进行的操作。一般情况下不需要返回值。
    3. 表达式：为了得到返回值的计算式。一定会有返回值，比如赋值语句的等号右边可以放置各种表达式。
    // 这是一行赋值语句
// 等号右边 1 + 3 是一个表达式
var a = 1 + 3;

    语法

    1. 语句以分号结尾。
    2. 表达式不需要分号结尾。一旦添加分号就会视为语句，这样会产生没有意义的语句。
    3. 分号前面没有内容会视为空语句。
    4. 一行内可以写多个语句，但规范写法应该一行一个语句。
    1 + 3   // 表达式
1 + 3;  // 语句
;;;     // 3个空语句
var a = 1 + 3; var b = 'abc'; // 1行内2个语句

    注释

    • 定义：源码中被忽略的部分，可以对代码进行解释说明。
    • 单行注释：使用 // 开头。
    • 多行注释：放在 /* 和 */ 之间。
    // 单行注释
/*
 * 多行注释
 * 多行注释
**/

    区块

    • 定义：使用大括号，将多个相关的语句组合到一起，成为区块。
    • 区块对于 var 命令不构成单独的作用域，与不使用区块的情况没有任何区别。
    • 区块一般不会单独使用，而是用来构成其它更复杂的语法结构，比如 if for 等。
    {
    var a = 1;  // 代码在区块内部
}
console.log(a); // 1 变量在区块外依然有效

    变量

    定义

    • 变量是存储信息的容器。变量是对“值”的具名引用。变量就是为“值”起名，然后引用这个名字，就等同于引用这个值。

    语法

    1. 使用关键字 var 声明变量，以分号结尾。
    2. var 可以省略，但规范写法禁止省略。
    3. 一个 var 命令可以声明多个变量，以逗号分隔。
    4. 重复声明一个已经存在的变量，没有赋值时是无效的，有赋值时会覆盖前面的值。
    5. 变量是弱类型的，没有限制，声明时可以初始化为任意值，也可以随时更改。
    6. 声明过的变量没有赋值，该变量的值是 undefined。
    7. 未声明的变量直接使用，系统会报错 not defind。
    var a = 1;      // 等同于 a = 1
var b, c, d;    // 多个变量
var e = 1;
var e;
console.log(e); // 1 重复声明没有赋值 无效
var e = 2;
console.log(e); // 2 重复声明有赋值 覆盖前面的值
var f = 1;      // 声明类型 number
    f = '1';    // 更改类型 string
var g;          // undefined
console.log(h); // h is not defined

    变量提升

    • 定义：JavaScript 引擎的工作方式是，先解析代码，获取所有已经声明的变量，再逐行运行。这样所有变量的声明语句，都会被提升到代码的头部，这叫做变量提升。
    • 示例：书写过程先使用变量，再声明赋值，这是是错误的做法，但实际并不会报错。因为存在变零提升，运行时会先声明变量，再使用，再赋值。所以返回值为 undefined 而不是 not defined。
    // 书写过程
console.log(a); // undefined 而不是 not defined
var a = 1;
// 运行过程
var a;
console.log(a);
a = 1;

    语句

    条件语句

    if 结构

    • if 后跟条件表达式，写在圆括号内，表示对表达式求值，根据求值结果执行不同的语句。
    • 求值结果的布尔值为 true 表示满足条件，执行紧跟在后面的语句，false 则跳过。
    • 多个语句必须写在大括号里表示区块，一个语句可以省略大括号。
    if (a === 1) {
    a = a + 1;          // 满足条件时执行
}
if (a === 1) a = a + 1; // 省略大括号

    if...else 结构

    • if 语句后跟 else 代码块，表示不满足条件时执行的代码。
    • else 代码块总是匹配最近的 if 语句。
    • 多个 if...else 语句连在一起使用，可以多次判断同一个变量。
    if (m === 3) {
    // 满足条件时执行
} else {
    // 不满足条件时执行
}
if (m === 0) {
    // ...
} else if (m === 1) {
    // ...
} else if (m === 2) {
    // ...
} else {
    // ...
}

    switch 结构

    • switch 和 case 后跟变量、表达式等。
    • switch 后的值如果与 case 后的值相等，会执行对应的 case，都不相等会执行 default。
    • case 内部需要添加 break 跳出 switch 结构。
    • switch 相当于多个 if...else 的简便形式。
    switch (fruit) {
    case 'banana':
        // fruit === 'banana' 时执行
        break;
    case 'apple':
        // ...
        break;
    default:
        // ...
}

    三元运算符

    • 三元运算符是 if...else 的简写形式。
    • 三元运算符是表达式，有返回值。
    • if...else 是语句，没有返回值。
    // 三元运算符
var even = (n === 0) ? true : false;
// 等同于
var even;
if (n === 0) {
    even = true;
} else {
    even = false;
}

    循环语句

    for 循环

    • for 语句写法与 if 语句类似。
    • for 后跟三个表达式，写在圆括号里。
      1. 初始化表达式：确定循环变量的初始值，只在循环开始时执行一次。
      2. 条件表达式：每轮循环开始时，都要执行这个条件表达式，只有值为真，才继续进行循环。
      3. 递增表达式：每轮循环的最后一个操作，通常用来递增循环变量。
    for (var i = 0; i < 3; i++) {
    console.log(i); // i < 3 时执行 循环3次输出 0 1 2 最后 i === 3
}
for (var i = 0; i < 3; i++) console.log(i); // 省略大括号

    while 循环

    • while 语句写法与 if 语句相同。
    • for 循环都可以改写成 while 循环。
    var i = 0;
while (i < 3) {
    console.log(i); // i < 3 时执行 循环3次输出 0 1 2 最后 i === 3
    i++;
}
while (i < 3) i++; // 省略大括号

    do...while 循环

    • do...while 循环与 while 循环类似。
    • do...while 先运行一次循环语句，再判断循环条件。不管条件是否为真都至少运行一次。
    • while 后的分号不能省略。
    var i = 0;
do {
    console.log(i); // i < 3 时执行 先执行1次再循环2次输出 0 1 2 最后 i === 3
    i++;
} while (i < 3);
do
    i++; // 省略大括号
while (i < 3);

    break 和 continue

    • break 跳出代码块或循环。
    • continue 立即终止本轮循环，返回循环结构的头部，开始下一轮循环。
    for (var i = 0; i < 5; i++) {
    console.log(i);     // 输出 0 1 2 3
    if (i === 3) break; // i 为 3 时跳出循环
}
    for (var i = 0; i < 5; i++) {
    if (i % 2 === 0) continue;  // i 为偶数时跳到下一轮循环
    console.log(i);             // i 为奇数时输出 1 3
}

    标签（label）

    • 语句的前面可以添加标签，相当于定位符，用于跳转到程序的任意位置。
    • 标签可以是任意的标识符，但不能是保留字，语句部分可以是任意语句。
    • 多重循环中，break 和 continue 只针对内层循环，使用标签可以跳出特定的循环。
    // 跳出区块 以下输出结果为 1 2
foo: {
    console.log(1);
    break foo;
    console.log('本行不会输出');
}
console.log(2);
// 跳出多重循环
egBreak:
    for (var i = 0; i < 3; i++) {
        for (var j = 0; j < 3; j++) {
            if (i === 1 && j === 1) break egBreak;  // i 为 1 且 j 为 1 时跳出
            console.log(i + '-' + j);               // 输出 0-0 0-1 0-2 1-0
        }
    }
// 跳出多重循环
egContinue:
    for (var i = 0; i < 3; i++) {
        for (var j = 0; j < 3; j++) {
            if (i === 1 && j === 1) continue egContinue;    // i 为 1 且 j 为 1 时跳到下一轮
            console.log(i + '-' + j);                       // 输出 0-0 0-1 0-2 1-0 2-0 2-1 2-2
        }
    }

    运算符

    运算顺序

    • 优先级：优先级高的运算符先执行。
    • 圆括号：() 可以提高运算的优先级。它不是运算符，而是一种语法结构。
    • 左结合：优先级相同的运算符，大多数情况，计算顺序总是从左到右。
    • 右结合：赋值运算符 =、指数运算符 **、三元条件运算符 ?: 从右边开始计算。
    • 特殊值：NaN 与任何值比较，都返回 false，包括它自己。
    1 + 2 * 3;     // 7 先乘后加
(1 + 2) * 3;   // 9 先加后乘
1 > NaN;       // false
NaN == NaN;    // false

    算术运算符

    算术运算符有 10 个，用于数学运算。

    类型 格式 说明 示例

    加法

    		 x + y
    • 加法运算符从左到右依次运算。
    • 加法运算符根据运算子的不同，执行不同的语法行为，比如相加或连接，这叫做重载。
    • 布尔值转换成数值相加。
    • 字符串最近的加法运算符变成连接符。
    		 
    1 + 1;         // 2 数值相加
1 + true;      // 2 布尔值相加
'1' + '2';     // '12' 字符串相加
'1' + true;    // '1true'
'1' + 2 + 3;   // '123'
1 + 2 + '3';   // '33'

    减法

    		 x - y
    • 不会发生重载。
    • 运算子先转为数值，再进行相应的数学运算。
    		 
    1 - '2'; // -1
1 * '2'; // 2
1 / '2'; // 0.5

    乘法

    		 x * y

    除法

    		 x / y

    指数

    		 x ** y
    • 前一个运算子是底数，后一个运算子是指数。
    • 右结合：多个指数运算符从右边开始计算。
    		 
    2 ** 4;         // 16
2 ** 2 ** 3;    // 256 等同于 2 ** (2 ** 3)

    余数

    		 x % y
    • 运算结果的正负号由第一个运算子决定。
    • 浮点数的运算无法得到完全准确的结果。
    		 
    -5 % 2;    // -1
5 % -2;    // 1
5.1 % 2.1; // 0.8999999999999995

    自增

    		 x++ 或 ++x
    • 一元运算符，运算子先转为数值，再加1或减1。
    • 运算符在变量后会先返回值再加减，变量前会先加减再返回值。
    • 运算的副作用：运算后原始变量的值会改变。
    		 
    var x = 1;
x++;   // 1
x;     // 2
var y = 1;
++y;   // 2
y;     // 2

    自减

    		 x-- 或 --x

    数值

    		 +x
    • 数值运算符可以将任何值转为数值，同 Number()。
    • 运算后返回一个新值，不改变原始变量的值。
    		 
    -true; // -1
+true; // 1
+[];   // 0
+{};   // NaN

    负数值

    		 -x

    比较运算符

    比较运算符有 4 个，与相等运算符一样，用于比较两个值的大小，然后返回一个布尔值，表示是否满足指定的条件。它可以比较各种类型的值，不仅仅是数值。

    类型 符号 说明 示例

    大于

    		 >
    1. 字符串：按照字典顺序，比较 Unicode 码点大小。
    2. 非字符串的原始类型：先调用 Number() 转成数值，再进行比较。
    3. 非字符串的对象类型：先调用 valueOf() 获取原始值，再进行比较。
      如果返回值还是对象，再调用 toString() 转成字符串，再进行比较。
    		 
    // 1.字符串
'cat' > 'Cat'; // true  c-99 > C-67
'大' > '小';   // false 大-22823 < 小23567
// 2.原始类型
5 > '4';  // true 等同于 5 > Number('4')
2 > true; // true 等同于 2 > Number(true)
// 3.对象
var x = [2]; // x.valueOf() == [2]; x.toString() == '2';
x > '11';    // true  等同于 '2' > '11'
x.valueOf = function() { return '1' }; // x.valueOf() == '1';
x > '11';    // false 等同于 '1' > '11'

    小于

    		 <

    小于或等于

    		 <=

    大于或等于

    		 >=

    相等运算符

    相等运算符有 4 个，与比较运算符一样，用于比较两个值的大小，然后返回一个布尔值，表示是否满足指定的条件。它可以比较各种类型的值，不仅仅是数值。

    类型 符号 说明 示例

    相等

    		 ==
    1. 相同类型：与严格相等运算符完全一样。
    2. 不同的原始类型：先转成数值，再进行比较。
    3. 对象与数值：对象转为数值。
    4. 对象与字符串：对象转为字符串。
    5. 对象与布尔值：对象和布尔值都转成数值。
    6. undefined 和 null 相互比较为 true，与其它值比较为 false。
    7. 相等运算符隐藏的类型转换，很容易出错，应该只使用严格相等运算符。
    		 
    // 2.不同的原始类型
true == 1;      // true  等同于 Number(true) === 1
true == 'true'; // false 等同于 Number(true) === Number('true') 即 1 === NaN
// 3.对象与数值
[1] == 1;       // true  等同于 Number([1]) == 1
// 4.对象与字符串
[1] == '1';     // true  等同于 String([1]) == '1'
// 5.对象与布尔值
[1] == true;    // true  等同于 Number([1]) == Number(true)
// 6.undefined 和 null
0 == null;      // false
0 == undefined; // false
undefined == null; // true

    严格相等

    		 ===
    1. 不同类型：直接返回 false。
    2. 相同的原始类型：值相同就返回 true，否则返回 false。
    3. 相同的对象类型：比较是否引用同一个内存地址，而不是值。
    4. 不同的变量引用同一个对象时，两个变量相等。
    5. undefined 和 null 与自身严格相等。
    		 
    // 1.不同类型
1 === "1";       // false
true === "true"; // false
// 2.相同的原始类型
1 === 0x1;       // true 十进制与十六进制值相同
// 3.相同的对象
[] === [];       // false
{} === {};       // false
function() {} === function() {}; // false
// 4.变量引用同一个对象
var v1 = {};
var v2 = v1;
v1 === v2;       // true
// 5.undefined 和 null
undefined === undefined; // true
null === null;   // true

    不相等

    		 != 返回相等/严格相等运算符运算结果的相反值。 
    1 != '1';  // false 等同于 !(1 == '1')
1 !== '1'; // true  等同于 !(1 === '1')

    严格不相等

    		 !==

    布尔运算符

    包括 3 个逻辑运算符和 1 个条件运算符，用于将表达式转为布尔值。

    类型 符号 说明 示例

    且 and

    		 &&
    • 用于多个表达式的求值，从左到右依次运算。
    • 如果表达式的布尔值为 false ，返回表达式的值且不再对后面的表达式求值。
    • 如果表达式的布尔值为 true，继续求值。
    • 如果所有表达式的布尔值都为 true，返回最后表达式的值。
    		 
    0 && 1;        // 0 第一个是false
1 && 0;        // 0 第二个是false
1 && 2;        // 2 都是true
1 && 2 && 3;   // 3 都是true

    或 or

    		 ||
    • 用于多个表达式的求值，从左到右依次运算。
    • 如果表达式的布尔值为 true，返回表达式的值且不再对后面的表达式求值。
    • 如果表达式的布尔值为 false ，继续求值。
    • 如果所有表达式的布尔值都为 false，返回最后表达式的值。
    • 或运算符常用于为一个变量设置默认值。
    		 
    1 || 0;            // 1 第一个是true
0 || 1;            // 1 第二个是true
0 || '';           // '' 都是false
0 || '' || null;   // null 都是false
x = res.x || 1;    // 先取返回值，没有时设置默认值1

    否 not

    		 !
    • 用于将布尔值转为相反值，非布尔值会先转为布尔值。
    • 以下六个值取反后为 true，其它所有值都为 false。
      1. undefined
      2. null
      3. false
      4. 0
      5. NaN
      6. 空字符串（''）
    • 两次取反可以转为对应的布尔值，与 Boolean() 相同。
    		 
    !undefined  // true
!null       // true
!false      // true
!0          // true
!NaN        // true
!""         // true
!true       // false
!1          // false
!'1'        // false
![]         // false
!!x         // 等同于 Boolean(x)

    三元运算符

    		 ?:
    • 语法：(条件) ? 表达式1 : 表达式2
    • 右结合：多个条件时从右边开始计算。
    • 条件为 true 返回表达式1的值，否则返回表达式2的值。
    		 
    
1 ? 'yes' : 'no';               // yes
0 ? 'yes' : 'no';               // no
0 ? 'yes' : 0 ? 'yes' : 'no';   // no 等同于下面的运算
0 ? 'yes' : (0 ? 'yes' : 'no'); // no

    二进制位运算符

    二进制位运算符有 7 个，用于直接对二进制位进行计算。它直接处理每一个比特位，是非常底层的运算，优点是速度极快，缺点是很不直观，使代码难以理解且容易出错。

    类型 符号 说明 示例
    运算规则
    1. JavaScript 会自动把64位浮点数，转成32位整数，再进行运算。
    2. 大于32位的数位会舍去。
    3. 小数部分会舍去。
    4. 逐位比较每个二进制位。
    		 
    0 | 3.1;  // 3 二进制 00 和 11 结果 11 小数部分会舍去

    二进制且 and

    		 & 逐位比较两个二进制位，存在 0 则结果为 0，否则为 1。 
    0 & 3;    // 0 二进制 00 和 11 结果 00
2 & 3;    // 2 二进制 10 和 11 结果 10
4 & 3;    // 0 二进制 100 和 11 结果 000

    二进制或 or

    		 | 逐位比较两个二进制位，存在 1 则结果为 1，否则为 0。 
    0 | 3;    // 3 二进制 00 和 11 结果 11
2 | 3;    // 3 二进制 10 和 11 结果 11
4 | 3;    // 7 二进制 100 和 11 结果 111

    二进制否 not

    		 ~ 将每个二进制位都变为相反值。自身与取反值相加为 -1。 
    ~ null;   // -1
~ -3;     // 2
~~ 3;     // 3 连续取反得到自身值

    异或 xor

    		 ^ 逐位比较两个二进制位，不相同则结果为 1，否则为 0。 
    0 ^ 3;    // 3 二进制 00 和 11 结果 11
2 ^ 3;    // 1 二进制 10 和 11 结果 01
4 ^ 3;    // 7 二进制 100 和 11 结果 111

    左移 left shift

    		 << 将一个数的二进制值向左移动 N 位数，尾部补 0，即 * 2N。 
    2 << 1;   // 4 二进制 10 结果 100
4 << 1;   // 8 二进制 100 结果 1000
-4 << 1;  // -8

    右移 right shift

    		 >> 将一个数的二进制值向右移动 N 位数，头部补 0，即 / 2N。 
    2 >> 1;   // 1 二进制 10 结果 1
4 >> 1;   // 2 二进制 100 结果 10
-4 << 1;  // -2

    右移（带符号位）
    zero filled right shift

    		 >>> 包括符号位的右移运算符。正数完全一致，负数有区别，运算结果总是为正。 
    4 >>> 1;  // 2
-4 >>> 1; // 2147483646

    其它运算符

    类型 符号 说明 示例

    赋值运算符

    		 x = y
    • 将等号右边的值赋给左边。
    • 等号可以和其它运算符结合，形成变体。
    • 复合的赋值运算符，先进行指定运算，再将返回值赋给左边的变量。
    • 右结合：多个赋值运算符从右边开始计算。
    		 
    var x = 1;
x = 2;  // 2
x += 2; //  3 即 x = x + 2
x -= 2; // -1 即 x = x - 2
x *= 2; //  2 即 x = x * 2
var y = 2;
var z = 3;
x = y = z;  // 3 等同于 x = (y = z)

    逗号运算符

    		 ,
    • 执行两个表达式，并返回后一个表达式的值。
    • 主要用于在返回一个值之前，进行一些辅助操作。
    		 
    var x = 0;
var y = (x++, 10);        // 先计算x为1，再计算y为10
var z = (alert(1), true); // 先弹出1，再计算z为true

    void 运算符

    		 void
    • 执行一个表达式，并返回 undefined。
    • 优先级很高，表达式应该总是写在圆括号内，否则容易出错。
    • 主要用于在超链接中插入代码防止网页跳转。
    		 
    void(0);     // undefined
void(1 + 2); // undefined
void 1 + 2;  // NaN 等同于 void(1) + 2 即 undefined + 2
    <a href="javascript:void(0)">防止跳转</a>
