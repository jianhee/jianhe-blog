    数据类型

    2个特殊值

    null

    &

    undefined

    对比

    • null 表示空值。
    • undefined 表示未定义或不存在，没有值。
    • undefined 由 null 派生，ECMAScript 定义它们相等，但不严格相等。
    • null 转换数值为 0。
    • undefined 转换数值为 NaN。
    var a = null;
var b = undefined;
null == undefined;  // true
null === undefined; // false
Number(null);       // 0
Number(undefined);  // NaN

    用法

    1. null 被认为是对象的占位符，使用 typeof 返回 object。
    2. 变量已声明，但未赋值或初始化，默认值是 undefined。
    3. 变量未声明，直接使用会报错，typeof 返回 undefined。
    4. 函数无返回值，返回 undefined。
    5. 函数缺少参数，参数值 undefined。
    6. 对象属性不存在，该属性值 undefined。
    typeof null;        // object
var b;
b;                  // undefined 未赋值
c;                  // 报错
typeof c;           // undefined 未声明
function g() {}
g();                // undefined 函数无返回值
function f(x) {
return x;
}
f();                // undefined 函数缺少参数
var  o = new Object();
o.p;                // undefined 对象属性不存在

    3个原始类型

    boolean

    • 布尔值 只有两个值 true 和 false，表示真伪，也叫逻辑值。

    • 以下运算符会返回布尔值：

      1. 比较运算符：>，>=，<，<=
      2. 相等运算符：===，!==，==，!=
      3. 逻辑运算符否： !
    var isTrue = true;
if (isTrue == true) {
// do something
}

    number

    数值表示法

    • 整数和浮点数

      1. 所有数字都是以64位二进制浮点数形式储存，即使整数也是如此。
      2. 浮点数不是精确的数字，涉及小数的运算要避免出错。
      0.3 / 0.1 === 3;     // false 结果为 2.9999999999999996
(0.3 - 0.2) === 0.1; // false 结果为 0.09999999999999998

    • 科学计数法：

      1. 写法为 e/E 后跟整数，表示 *10整数。
      2. 小数点前的数字多于21位时。
      3. 小数点后的零多于5位时。
      5e2;  // 500  等同于 5*10²
5e-2; // 0.05 等同于 5*10¯²
1234567890123456789012; // 1.2345678901234568e+21 22位整数
0.0000003; // 3e-7 小数点后紧跟6个0

    • 进制：

      1. 十进制：没有前导0的数值。
      2. 二进制：有前缀 0b/0B 的数值。
      3. 十六进制：有前缀 0x/0X 的数值。
      4. 八进制：有前缀 0o/0O 的数值，或者有前导0后跟7及以内数字的数值（已废除）。
      5. JavaScript 内部会自动将八进制、十六进制、二进制转为十进制。
      3;     // 3 十进制值
0b11;  // 3 二进制值
0xff;  // 255 十六进制值
0o377; // 255 八进制值
0777;  // 511 八进制值 前导0后跟7以内 易混乱已废除
0888;  // 888 十进制值 前导0后跟7以外

    特殊值

    • 0

      1. +0 和 -0 是等价的，都可以表示为 0。
      2. 0 除以 0 得到 NaN。
      3. 非 0 数值除以 0，得到 Infinity/-Infinity，正负号由数值和 0 共同决定。
      +0 === -0;  // true
0 / 0;      // NaN
1 / 0;      // Infinity
1 / -0;     // -Infinity
-1 / -0;    // Infinity

    • NaN

      1. NaN 表示非数字。
      2. NaN 主要出现在将字符串解析成数字出错的场合，或者数学运算出错时。
      3. NaN 与所有值都不相等，与任何值比较，都返回 false，包括它自己。
      4. NaN 不是独立的数据类型，而是一个特殊数值。
      1 + 'x';        // NaN
Math.log(-1);   // NaN
Number('abc');  // NaN
NaN === NaN;    // false
typeof NaN;     // 'number'

    • Infinity

      1.  Infinity 表示正无穷，大于一切数值。
      2. -Infinity 表示负无穷，小于一切数值。
      3. Infinity 与 NaN 比较，返回 false。
      4. Infinity 的运算规则了解即可。
      Math.pow(2, 1024); // Infinity 运算结果超出最大值
Infinity > 1000;   // true
-Infinity < -1000; // true
-Infinity < NaN;   // false

    • 其它特殊值：

      1. Number.MAX_VALUE 最大有效值
      2. Number.MIN_VALUE 最小有效值
      3. Number.POSITIVE_INFINITY 正无穷大
      4. Number.NEGATIVE_INFINITY 负无穷大
      Number.MAX_VALUE; // 1.7976931348623157e+308
Number.MIN_VALUE; // 5e-324
Number.POSITIVE_INFINITY; // Infinity
Number.NEGATIVE_INFINITY; // -Infinity

    数值组成

    • 第1位：符号位，0 表示正数，1 表示负数。
    • 第2位到第12位（共11位）：指数部分。
    • 第13位到第64位（共52位）：小数部分（即有效数字）。
    • 公式：(-1)符号位 * 1.xx...xx * 2指数部分

    数值精度

    • 小数部分决定数值精度。
    • 小数部分的第一位默认总是1，不保存在64位浮点数之中，所以有效数字最长为52+1=53个二进制位。
    • 精度最多为53个二进制位，即-253到253之间都可以精确计算，超出范围开始出错。
    • 253是16位的十进制数值，所以15位的十进制数值都可以精确计算。
    Math.pow(2, 53);     // 9007199254740992
Math.pow(2, 53) + 1; // 9007199254740992 开始出错
Math.pow(2, 53) + 2; // 9007199254740994
Math.pow(2, 53) + 3; // 9007199254740996
Math.pow(2, 53) + 4; // 9007199254740996
9007199254740992111; // 9007199254740992000 多出的三个有效数字，将无法保存

    数值范围

    • 指数部分决定数值大小。
    • 指数部分最长为11个二进制位，最大值为211-1=2047。
    • 正数1024，负数1023，再加上52位小数部分，即21024到2-1075之间都可以正确表示。
    • 正向溢出：大于最大值，返回 Infinity。
    • 负向溢出：小于最小值，返回 0。
    Math.pow(2, 1024);   // Infinity 正向溢出
Math.pow(2, -1075);  // 0 负向溢出

    string

    定义

    • 字符串 是零个或多个字符的集合，放在单引号或双引号内，表示文本。
    • 字符可以是任意文本，包括字母、数字、符号等，也可以为空。
    'abc';  // 字母
'123';  // 数值
'&';    // 符号
' ';    // 空

    换行

    1. 字符串默认只能写在一行内，分成多行将会报错。
    2. 使用反斜杠 \ 写在行尾可以换行。但反斜杠后面不能有任何字符，包括空格。
    3. 使用连接运算符 + 可以连接多个单行字符串。
    'a
b
c'      // Uncaught SyntaxError: Invalid or unexpected token
'a\
b\
c'      // abc
'a' +
'b' +
'c'     // abc

    属性

    1. length 属性返回字符串的长度，但该属性无法改变。
    2. 使用数组的方括号运算符，可以返回某个位置的字符。
    var s = 'hello';
s.length; // 5
s[0];     // h

    转义字符

    1. \\ 反斜杠
    2. \' 单引号
    3. \" 双引号
    4. \n 换行符
    'a \\ b'; // 'a \ b'
'a \n b'; // 'a' // 'b' 输出两行

    字符集

    • Unicode

      1. JavaScript 使用 Unicode 字符集，所有字符都用 Unicode 表示和储存。
      2. JavaScript 还允许直接使用 Unicode 码点表示字符，即 \uxxxx 形式。
      3. JavaScript 解析代码时，会自动识别字符的表示形式，但输出时会转成字面形式。
      var s = '\u00A9';          // 使用码点表示字符 输出字面 '©'
var f\u006F\u006F = 'abc'; // 使用码点声明变量
foo;                       // 使用对应的字面可以输出变量值 'abc'

    • UTF-16

      1. JavaScript 内部使用 UTF-16 格式储存字符。
      2. 码点在 U+0000 到 U+FFFF 之间的字符，长度为16位，即2个字节。
      3. 码点在 U+10000 到 U+10FFFF 之间的字符，长度为32位，即4个字节。
      4. 由于历史原因，JavaScript 只支持两字节的字符，不支持四字节的字符，四字节会被认为是两个字符，即 length 为 2。所以 JavaScript 返回的字符串长度可能是不正确的。

    • Base64

      1. Base64 是一种编码方法，可以将任意值转成 0-9、A-Z、a-z、+ 和 / 这64个字符组成的可打印字符。
      2. 使用 Base64 编码可以避免出现特殊字符，简化程序的处理。
      3. 场景1：文本包含不可打印的符号，比如 ASCII 码0到31的符号，可以使用 Base64 编码转成可以打印的字符。
      4. 场景2：以文本格式传递二进制数据，也可以使用 Base64 编码。

    1个合成类型

    object

    • 定义：各种值的集合。对象是拥有属性和方法的数据集合，可以看做是存放各种值的容器。JS 中的所有事物都是对象，包括字符串、数字、数组、日期等。
    • 属性：与对象相关的值。
    • 方法：能够在对象上执行的动作。
    • 成员：在面向对象的语言中，属性和方法常被称为对象的成员。
    var car;                // 汽车是现实生活中的对象
car.color = white;      // 汽车的属性
car.start();            // 汽车的方法

    方法

    数据类型检测

    typeof 运算符

    • 定义：检测一个值的数据类型。

    • 返回值：

      1. undefined 1.undefined 2.未赋值 3.未声明 4.函数无返回值
      2. boolean   布尔值
      3. number    数值
      4. string    字符串
      5. object    1.对象 2.null
      6. function  函数
    var g;
function f() {};
typeof undefined;   // undefined
typeof g;           // undefined 变量 g 未赋值
typeof h;           // undefined 变量 h 未声明
typeof f();         // undefined 函数 f 无返回值
typeof true;        // boolean
typeof 123;         // number
typeof '123';       // string
typeof [];          // object
typeof {};          // object
typeof null;        // object
typeof f;           // function f 是一个函数

    数值相关

    isNaN()

    • 定义：判断一个值是否为 NaN 非数值。
    • 返回值：非数值返回 true，数值返回 false。
    • 说明：非数值先转为数值再判断，所以字符串、对象等也可能返回 true，使用前最好判断一下数据类型。
    isNaN(NaN);     // true
isNaN('a');     // true  字符串
isNaN(['a']);   // true  数组
isNaN({});      // true  对象
isNaN(1);       // false 数值
isNaN('1');     // false 字符串转为1
isNaN([]);      // false 空数组转为0
isNaN([1]);     // false 数组只有一个数值成员 转为1
isNaN(['1']);   // false
typeof value === 'number' && isNaN(value);

    isFinite()

    • 定义：判断一个值是否是有穷的数值。
    • 返回值：有穷返回 true，无穷返回 false。

    • 类型：

      1. 非数值返回 false。
      2. Infinity/-Infinity、NaN、undefined 返回 false。
      3. 其它数值都返回 true。
    isFinite('a');          // false 非数值
isFinite(Infinity);     // false
isFinite(-Infinity);    // false
isFinite(NaN);          // false
isFinite(undefined);    // false
isFinite(null);         // true
isFinite(-1);           // true

    parseInt()

    &

    parseFloat()

    parseInt() parseFloat() 示例
    返回值
    • 将字符串转为十进制整数。
    • 小数点是无效字符。
    • 将字符串转为十进制小数。
    • 第一个小数点有效，后面的小数点是无效字符。
    		 
    parseInt('1.23');       // 1
parseFloat('1.23');     // 1.23
parseFloat('1.2.3');    // 1.2
    转换
    • 非字符串会先转为字符串再转换。
    • 字符串头部的空格会被自动去除。
    		 
    parseInt(123);     // 等同于 parseInt('123')
parseInt('  123'); // 123
    规则
    • 从第一个字符开始，依次解析转换。
    • 如果字符能转为数值，则继续解析，否则停止解析，并返回已经转好的部分。
    • 如果第一个字符不能转为数值，则返回 NaN。
    • 如果第一个字符是正负号，并且后跟数值，会继续解析。
    • 科学计数法会继续解析。
    		 
    parseInt('123a');   // 123
parseInt('123*');   // 123
parseInt('a123');   // NaN
parseInt('abcd');   // NaN
parseInt('');       // NaN
parseInt('+');      // NaN
parseInt('+123');   // 123
    科学计数法
    • 会自动转为科学计数法的数值，先转为以科学计数法表示的字符串再解析。
    • parseInt() 不会进行相应的转换，只解析有效字符。
    • parseFloat() 会进行相应的转换，再解析转换后的数值。
    		 
    parseInt(0.0000008);    // 等同于 parseInt('8e-7')
parseInt('123e-2');     // 123
parseFloat('123e-2');   // 1.23
    进制
    • 字符串可以用其它进制的数表示。
    • 比如 0x/0X 开头，按照十六进制数解析。
    • 字符串必须以十进制的数表示。
    • 前导会按照规则解析。
    		 
    parseInt('0x10');   // 16
parseFloat('0x10'); // 0 x是无效字符
    基模式
    • 第二个参数表示被解析的值的进制，默认为10。
    • 参数可设置为2-36之间的整数，超出范围返回 NaN。
    • 参数是 0、null、undefined，会直接忽略。
    • 指定进制的数不包含的字符也是无效的。
    		 没有第二个参数，字符串按照十进制数解析。 
    parseInt('10');        // 10  默认
parseInt('10', 2);     // 2   按照二进制数转换
parseInt('10', 50);    // NaN 参数超出范围
parseInt('10', null);  // 10  参数被忽略
parseInt('123', 2);    // 1   二进制数只有01有效
parseInt('312', 2);    // NaN

    转码

    btoa()

    &

    atob()

    • btoa() 将任意值转为 Base64 编码。
    • atob() 将 Base64 编码转为原来的值。
    • 非 ASCII 码的字符使用这两个方法，中间必须插入一个转码环节，否则会报错。
    var string = 'Hello World!';
btoa(string);               // 'SGVsbG8gV29ybGQh'
atob('SGVsbG8gV29ybGQh');   // 'Hello World!'
btoa('你好');               // 报错
atob('你好');               // 报错

    对象实例

    new Object()

    • 构建对象：由运算符 new 加上要实例化的对象的名字来创建。
    • 字面量：由花括号声明，花括号内，对象的属性以键值对 key/value 的形式来定义，由逗号分隔。
    • 访问对象属性：使用 . 或 []。

    1.构建对象 再赋值

    var person = new Object();
person.firstname = 'Bill';
person.lastname = 'Gates';
person.id = 5566;

    2.构建对象 同时赋值

    var person = new Object({
firstname: 'Bill',
lastname: 'Gates',
id: 5566
});

    3.对象字面量 直接赋值

    var person = {
firstname: 'Bill',
lastname: 'Gates',
id: 5566
};

    4.访问属性

    name = person.lastname;     // Gates
name = person['lastname'];  // Gates

    new Array()

    • 定义：Array 对象用于在单个变量中存储多个值。
    • 字面量：由中括号声明，中括号内的元素由逗号分隔。
    • 访问数组元素：使用 [] 和数组元素下标，0 是第一个元素。

    1.构建数组 再赋值

    var cars = new Array();
cars[0]='Audi';
cars[1]='BMW';
cars[2]='Volvo';

    2.构建数组 同时赋值

    var cars = new Array('Audi','BMW','Volvo');

    3.数组字面量 直接赋值

    var cars = ['Audi','BMW','Volvo'];

    4.访问数组元素

    cars[1];        // BMW

    new Date()

    • 定义：Date 对象用于处理日期和时间。
    • 参数：可以是 4 种。
      1. 空：返回当前日期和时间。
      2. 13 位毫秒数。
      3. 数字格式的日期和时间。
      4. 字符串格式的日期和时间。
    • 返回值：返回本地标准时间。
    • 方法：
      1. 必须结合一个 Date 对象来使用。
      2. get 根据本地时间返回值。
      3. set 根据本地时间设置值。
      4. UTC 根据世界时间返回或设置。
      5. 点击这里 查看全部。
    /* 创建 Date 对象的四种方法 */
new Date();                     // Tue Sep 18 2018 10:02:15 GMT+0800 (中国标准时间)
new Date(1536887640000);        // Fri Sep 14 2018 09:14:00 GMT+0800 (中国标准时间)
new Date(2018, 8, 14, 9, 14, 0);
new Date('2018, 9, 14 9:14:00');

/* Date 对象方法 以 2018-9-14 9:14:00 星期五 为例 */
var $date = new Date(2018, 8, 14, 9, 14, 0);
$date.getFullYear();                // 2018
$date.getMonth();                   // 8 实际月份 +1 为 9 月
$date.getDate();                    // 14
$date.getDay();                     // 5 星期五
$date.getHours();                   // 9
$date.getUTCHours();                // 1 UTC小时,相差 8h
$date.getMinutes();                 // 14
$date.getSeconds();                 // 0
$date.getMilliseconds();            // 0
$date.getTime();                    // 1536887640000
Date.parse('2018, 9, 14 9:14:00');  // 1536887640000
Date.UTC(2018, 8, 14, 9, 14, 0);    // 1536916440000 UTC毫秒,相差 8h
$date.getTimezoneOffset();          // -480 时区为 GMT+0800
$date.setHours(8);                  // 1536884040000 设置小时后的毫秒
console.log($date);                 // Fri Sep 14 2018 08:14:00 GMT+0800 (中国标准时间)

    Math

    • 定义：用于执行数学任务。
    • 注释：Math 对象没有构造函数 Math()，无需创建，可直接作为对象调用其属性和方法。
    • 属性示例：
      1. Math.PI 返回圆周率。
      2. 点击这里 查看全部。
    • 方法示例：
      1. Math.abs(x) 返回 x 的绝对值。
      2. Math.max(x,y) 返回 x 和 y 中的最大值。参数可以是多个值。
      3. Math.min(x,y) 返回 x 和 y 中的最小值。参数可以是多个值。
      4. Math.random() 返回 0 ~ 1 之间的随机数。
      5. Math.round(x) 返回 x 四舍五入后最接近的整数。
      6. 点击这里 查看全部。
    /* Math 对象属性 */
Math.PI;            // 3.141592653589793

/* Math 对象方法 */

Math.round(3.5);    // 4
Math.round(-3.5);   // -3

    原始类型构建对象

    • new Boolean() Boolean 对象表示两个值 true 或 false。
    • new Number() Number 对象是原始数值的包装对象。
    • new String() String 对象用于处理字符串。
    new Boolean(0);     // 构造函数 返回 Boolean {false}

    new String(6);      // 构造函数 返回 String {'6'}
String(6);          // 转换函数 返回 '6'

    对象方法

    强制转换

    Boolean()

    • 定义：把给定的值转换成 Boolean 型。
    • 返回值：
      1. false：空字符串、数字 0、NaN、undefined、null。
      2. true：非空字符串、非 0 数字、对象。
    Boolean('');        // false
Boolean(' ');           // true
Boolean('0');           // true
Boolean('123');         // true

    Number()

    • 定义：把给定的值转换成数字。
    • 不同数据类型的返回值：
      1. Undefined：返回 NaN。
      2. Null：返回 0。
      3. Boolean：返回 0 和 1。
      4. Number：返回原始值。
      5. String：对整体值判断，空返回 0，数字返回原始值，非数字值返回 NaN。

      7. Object：返回 NaN。
    Number(undefined);      // NaN
Number(null);           // 0
Number(false);          // 0

Number(['a']);          // NaN
Number([1, 2]);         // NaN
Number({});             // NaN

    String()

    • 定义：把给定的值转换成字符串。
    • 说明：String() 和 toString() 方法唯一的区别是 undefined 和 null 调用不会报错。
    var a = null;
a.toString();           // 报错
String(undefined);      // 'undefined'
String(null);           // 'null'

    转换解析

    toString()

    • 定义：把对象转换成字符串，并返回结果。
    • 说明：所有对象都有该方法，无论它是真对象还是伪对象。
    • 不同对象的返回值：
      1. Boolean：伪对象，返回原始值 true 或 false。
      2. Number：伪对象，默认模式 返回十进制数字值，基模式 返回指定的其他进制的数字值。
      3. String：伪对象，一定有该方法。
      4. Array：数组对象，返回数组中的元素，用逗号分隔。join() 方法没有参数时与该方法返回值相同。
      5. Date：返回本地时间。
    • 相关方法：
      1. toLocaleString() 转换成本地格式的字符串。
      2. toLocaleString() 和 toUTCString() 根据本地时间/世界时转换。
      3. toTimeString() 和 toDateString() 转换时间/日期部分。
      4. toLocaleTimeString() 和 toLocaleDateString() 根据本地时转换时间/日期部分。
    var isTrue = false;
isTrue.toString();      // 输出 'false';

var iNum1 = 10;
var iNum2 = 10.0;
var iNum3 = 10.1;
iNum1.toString();       // 输出 '10'   十进制形式
iNum2.toString();       // 输出 '10'   十进制形式小数 省略小数点后末尾的0
iNum3.toString();       // 输出 '10.1' 十进制形式小数
iNum1.toString(2);      // 输出 '1010' 二进制形式
iNum1.toString(8);      // 输出 '12'   八进制形式
iNum1.toString(16);     // 输出 'A'    十六进制形式

var arr = ['a','b','c']
arr.toString();         // 输出 a,b,c

    valueOf()

    • 定义：返回对象的原始值。
    • 说明：对于许多对象，该方法与 toString() 方法的返回值相同。

    join()

    • 定义：把数组对象转换成字符串。
    • 参数：可选。指定要使用的分隔符，省略时使用逗号，此时与 toString() 方法返回值相同。
    • 返回值：返回一个字符串，使用指定的分隔符分隔。
    var arr = [1, 2, 3];
arr.join();             // 1,2,3
arr.join('.');          // 1.2.3

    split()

    • 定义：把字符串分割成字符串数组。
    • 参数：
      1. 必需。指定要分割的位置，可以是字符串或正则表达式。
      2. 可选。指定要返回的长度，省略时整个字符串都会被分割并返回。
    • 返回值：返回一个数组，不包含分隔符。
    var str = 'a|b|c';
str.split('|');     // ['a', 'b', 'c']
str.split('|', 2);  // ['a', 'b']

    toFixed()

    • 定义：将 Number 四舍五入为指定小数位数的数字。
    • 参数：必需。规定小数的位数，0 ~ 20 之间。
    • 返回值：返回 Number 的字符串表示。
    var num = new Number(13.357);
num.toFixed(2);     // 13.36 保留 2 位小数

    隐式类型转换

    • 说明：js数据类型不严格，没有浮点和整型，隐式类型转换指字符串和数字之间的转换。
    • 转换方式：
      1. 加法运算：默认将数字看成字符串进行拼接。
      2. 减乘除取余运算 和 比较运算：自动把字符串转换为数字，转换数字的默认方法是调用Number()。
    var x = '2';
var y = 1;
alert(x + y);   // '21'

alert(x - y);   // 1
alert(x * y);   // 2
alert(x / y);   // 2
alert(x % y);   // 0

alert(x == y);  // false

    查找替换

    charAt()和charCodeAt()

    • 定义：检索字符串中指定位置的字符。
    • 参数：指定位置的字符在字符串中的下标。
    • 返回值：返回指定位置的 字符/字符的 Unicode 编码。
    var str = 'abc';
str.charAt(0);      // a
str.charCodeAt(0);  // 97

    indexOf()和lastIndexOf()

    • 定义：从头到尾/从尾到头 在字符串或数组中检索指定的值。
    • 参数：
      1. 必需。规定要检索的字符或元素。
      2. 可选。规定开始检索的位置。省略时从 第一个/最后一个 开始。
    • 说明：检索数组时，字符必须与某个数组元素完全相同，不能只是元素的一部分。
    • 返回值：返回检索内容在字符串或数组中 首次/最后一次 出现的位置，没有时返回 -1。
    var str = 'ababab';
str.indexOf('ab');         // 0
str.lastIndexOf('ab');     // 4
str.indexOf('ab',2)        // 2  从第三个开始检索
str.indexOf('AB');         // -1 大小写敏感

var arr = ['ab', 'ab', 'ab'];
arr.indexOf('ab');         // 0
arr.lastIndexOf('ab');     // 2
arr.indexOf('a');          // -1 必须与数组元素完全相同

    search()

    • 定义：在字符串中检索指定的值或与正则表达式匹配的值。
    • 参数：必需。规定要检索的值。可以是字符串或要匹配的 RegExp 对象。
    • 返回值：返回第一个相匹配的值的位置，不能全局检索，没有返回-1。
    var str = '123abc123abc';
str.search(/\d+/);     // 0  检索正则
str.search(/abc/);     // 3  检索字符串
str.search(/ABC/);     // -1 大小写敏感
str.search(/ABC/i);    // 3  添加 i 忽略大小写

    match()

    • 定义：在字符串或数组中检索指定的值或与正则表达式匹配的值。
    • 参数：与 search() 相同。
    • 返回值：返回一个数组，包含相匹配的值，没有时返回 null。
      1. 默认返回第一个相匹配的值和相关信息。可忽略相关信息直接当做值使用。
      2. 添加 g 时返回字符串中全部相匹配的值。
    var str = '123abc123abc';
str.match(/\d+/);      // ['123']        检索正则
str.match(/\d+/g);     // ['123', '123'] 全局检索正则
str.match(/abc/);      // ['abc'] 检索字符串
str.match(/ABC/);      // null    大小写敏感
str.match(/ABC/i);     // ['abc'] 添加 i 忽略大小写

    replace()

    • 定义：在字符串中替换指定的值或与正则表达式匹配的值。
    • 参数：
      1. 必需。规定要替掉的旧值。可以是字符串或要匹配的 RegExp 对象。
      2. 必需。规定要换上的新值。可以是字符串或生成字符串的函数。
    • 返回值：返回一个新的字符串。
      1. 默认替换字符串中第一个相匹配的值。
      2. 添加 g 时替换字符串中全部相匹配的值。
    var str = '123abc123abc';
str.replace('abc', '字母');  // '123字母123abc'
str.replace(/abc/g, '字母'); // '123字母123字母' 全局替换
str.replace(/\d+/, '数字');  // '数字abc123abc'
str.replace(/\d+/g, '数字'); // '数字abc数字abc'

    截取字符

    slice()

    • 定义：在字符串或数组中选取指定位置之间的值。
    • 参数：
      1. start：必需。规定开始位置（包括）。0 第一个，-1 最后一个。
      2. end：可选。规定结束位置（不包括）。0 第一个，-1 最后一个，空 选取到最后一个。
    • 返回值：返回新的字符串或数组，包含选取的指定的值。
    var str = '012345';
var arr = [0, 1, 2, 3, 4, 5];
str.slice(1);         // '12345' 从第1个开始选到最后
arr.slice(1);         // [1, 2, 3, 4, 5]
str.slice(-1);        // '5'     从最后1个开始选
arr.slice(-1);        // [5]
str.slice(1, 3);      // '12'    从第2个开始选取到第3个
arr.slice(1, 3);      // [1, 2]
arr.slice(1, -1);     // '1234'  从第2个开始选取到倒数第2个
arr.slice(1, -1);     // [1, 2, 3, 4]

    substring()

    • 定义：在字符串中选取指定位置之间的字符。
    • 参数：
      1. start：必需。规定开始位置（包括），非负数。0 第一个。
      2. end：可选。规定结束位置（不包括），非负数。0 第一个，空 选取到最后一个。
    • 返回值：返回新的字符串，包含选取的字符。
    var str = 'abcde';
str.substring(1);      // bcde 从第2个开始选到最后
str.substring(1, 2);   // b    从第2个开始选到第2个

    substr() 不建议使用

    • 定义：在字符串中选取指定数目的字符。
    • 参数：
      1. start：必需。规定开始位置（包括）。0 第一个，-1 最后一个。
      2. length：可选。规定选择数目。空 选取到最后一个。
    • 返回值：返回新的字符串，包含选取的字符。
    var str = 'abcde';
str.substr(1);      // bcde 从第2个开始选到最后
str.substr(1, 2);   // bc   从第2个开始选2个

    添加删除

    concat()

    • 定义：连接字符串或数组。
    • 参数：字符串或数组。
    • 返回值：返回一个新的字符串或数组，类型与原变量相同。
    var str1 = 'a';
var str2 = 'b';
var arr1 = [1, 2];
var arr2 = [3, 4];
str1.concat(str2);         // 字符串+字符串 'ab'
str1.concat(arr1);         // 字符串+数组   'a1,2'
arr1.concat(str1);         // 数组+ 字符串  [1, 2, "a"]
arr1.concat(arr2);         // 数组+ 数组    [1, 2, 3, 4]
arr1.concat(str1, arr2);   // 连接多个      [1, 2, "a", 3, 4]

    push()和unshift()

    • 定义：向数组的 末尾/开头 添加元素。unshift()不兼容IE浏览器。
    • 参数：要添加到数组的元素。
    • 返回值：改变后的新数组的长度。
    • 原数组：改变。
    var arr1 = [1, 2, 3];
var arr2 = [1, 2, 3];
console.log(arr1.push('a', 'b'));       // 5
console.log(arr1);                      // [1, 2, 3, 'a', 'b']
console.log(arr2.unshift('a', 'b'));    // 5
console.log(arr2);                      // ['a', 'b', 1, 2, 3]

    pop()和shift()

    • 定义：删除并返回数组的 最后一个/第一个元素。
    • 返回值：返回原数组的 最后一个/第一个元素。原数组为空则返回 undefined 值。
    • 原数组：改变，长度-1。
    var arr1 = [1, 2, 3];
var arr2 = [1, 2, 3];
arr1.pop();             // 3
arr2.shift();           // 1
console.log(arr1);      // [1, 2]
console.log(arr2);      // [2, 3]
var arr0 = [];
arr0.pop();             // undefined
arr0.shift();           // undefined
console.log(arr0);      // []

    splice()

    • 定义：在数组中添加或删除元素。
    • 参数：
      1. index：必需。要添加或删除元素的位置（包括该元素）。0 第一个，-1 最后一个。
      2. howmany：必需。要删除的元素数量。0 不会删除元素。
      3. item：可选。向数组添加的新元素。
    • 返回值：如果删除了元素，返回一个新数组，包含被删除的元素。
    • 原数组：改变，添加或删除后的数组。
    var arr1 = [0, 1, 2, 3, 4];
arr1.splice(0,0);           // 从第1个开始删除0个添加0个 []
console.log(arr1);          // 原数组不变 [0, 1, 2, 3, 4]
arr1.splice(1,2);           // 从第2个开始删除2个添加0个 [1, 2]
console.log(arr1);          // 原数组剩余3个 [0, 3, 4]
arr1.splice(1,0,'a','b');   // 从第2个开始删除0个添加2个 []
console.log(arr1);          // 原数组剩余7个 [0, 'a', 'b', 1, 2, 3, 4]
arr1.splice(1,2,'a','b');   // 从第2个开始删除2个添加2个 [1, 2]
console.log(arr1);          // 原数组剩余5个 [0, 'a', 'b', 3, 4]

    比较排序

    localeCompare()

    • 定义：用本地特定的顺序来比较两个字符串。
    • 返回值：返回说明比较结果的数字。例如比较 a 和 b：
      1. a < b 时，返回值 < 0，a 在 b 之前。
      2. a = b 时，返回值 = 0，a 和 b 并列。
      3. a > b 时，返回值 > 0，a 在 b 之后。
    var str = 'b';
str.localeCompare('a');    // 1 说明 b 在 a 之后
str.localeCompare('b');    // 0
str.localeCompare('c');    // -1 说明 b 在 c 之前
str.localeCompare('d');    // -1

    reverse()

    • 定义：颠倒数组中元素的顺序。
    • 原数组：改变。
    var arr1 = [1, 2, 3, 4];
arr1.reverse();             // [4, 3, 2, 1]

    sort()

    • 定义：对数组的元素进行排序。
    • 参数：可选。可以是：
      1. 省略：默认按字符编码排序。
      2. 比较函数：添加两个参数 ab，a < b 时， 返回值 < 0， a 在 b 之前。
    • 返回值：返回重新排序后的数组。
    • 原数组：改变。
    var arr1 = ['Jo', 'Th', 'Ja', 'Ad'];
var arr2 = ['10', '5', '100', '1'];
arr1.sort();            // 'Ad',  'Ja', 'Jo', 'Th'
arr2.sort();            // ['1', '10', '100', '5'] 字符编码排序
arr2.sort(sortNumber);  // ['1', '5', '10', '100'] 大小排序
function sortNumber(a, b) {
return a - b;
}
